Create widgets to display business contact information.

== Description ==

Create widgets to display business contact information. This is not a contact manager as business information is stored in each widget.

* Business name and address
* Up to three business phone numbers
* Link to business web site and contact form or email address

== Installation ==

1. Upload `business-block-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the default settings on the settings panel.
4. Add widgets to your sidebar.
